## Remediation Priority

| Priority | Criteria | Reason |
|----------|----------|--------|
| P0 — Block release | All Level A FAIL | WCAG compliance minimum |
| P1 — Fix this sprint | Level AA FAIL affecting core flows | User impact high |
| P2 — Fix next sprint | Level AA FAIL peripheral | User impact medium |
| P3 — Backlog | Level AAA / cosmetic | Enhancement only |
