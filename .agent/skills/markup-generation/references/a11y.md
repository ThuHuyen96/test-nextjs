# Accessibility

- **General:** WCAG-minded; do not duplicate ARIA already provided.
- **Forms:** label inputs; expose errors accessibly.
- **Buttons:** **SButton** (not `div` + click); meaningful visible text.
- **Images:** meaningful `alt`; `alt=""` when decorative.
- **Keyboard:** focusable controls; preserve tab order—no random `tabindex`.
- **Avoid:** `div` with `onClick` as primary control; unlabeled fields; arbitrary `tabindex`.
