# Semantic HTML

- **Regions:** `header` (top), `main` (primary content), `footer` (bottom).
- **Grouping:** `section` for grouped blocks; `article` for self-contained units.
- **Text:** `h1`–`h6` for headings; do not skip levels.
- **Lists:** `ul` / `ol`; do not fake lists with `div`.
- **Actions vs nav:** **SButton** for actions; **`<a>`** for navigation.
- **Avoid:** everything a `div`; clickable `div` instead of button; broken heading order.
